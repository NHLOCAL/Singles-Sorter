# מסדר הסינגלים
סקריפט שמסדר קבצי מוזיקה בודדים בתיקיות מסודרות לפי אמנים.

הדאטה של הקוד מכילה זמרים יהודים חרדים בלבד, וממילא היא מיועדת לציבור החרדי בישראל בלבד.
